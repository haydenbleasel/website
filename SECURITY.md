# Security Policy

## Supported Versions

The only supported version of this website is the `main` branch as there are no releases or packages.

## Reporting a Vulnerability

To report a vulnerability, please open an issue in this repository or, if you have a fix, open a pull request. As this is an open source project with low impact, feel free to disclose the vulnerability publicly before it has been fixed. If you do want to disclose it privately, please contact me on my [website](https://haydenbleasel.com/contact).
